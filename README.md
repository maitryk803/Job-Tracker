# 📋 Project Job Tracker

A simple, clean web app to track job applications and stay organized during your job search.

## 🚀 Features

- Add and manage job applications in one place
- Track the status of each application (Applied, Interview, Offer, Rejected)
- View all your applications at a glance
- Lightweight — no frameworks or installs needed

## 🛠️ Built With

- HTML
- CSS
- JavaScript

## 📁 Project Structure

project-job-tracker/
├── index.html      # Main page and structure
├── script.js       # App logic and interactivity
└── style.css       # Styling and layout

## 🖥️ How to Use

1. Clone or download this repository
2. Open `index.html` in your browser
3. Start adding your job applications!

No installations or dependencies required — runs entirely in the browser.
